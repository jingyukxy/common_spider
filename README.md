#核心库模型
###业务流程
```Spider->TagEngine->AppCluster->Api->App/WebSite->Nginx->CDN->USER```

###核心库
1. 广告模块
    * 广告模块，全局广告模块，存储提供JS或外部JS存储，并且需要提供统计功能
    1. 广告表
    2. 广告主表
    3. 广告js模板
    4. 广告版位表
    5. 统计数据表
2. 统计分析模块 
    * 统计广告，访问，点赞等等信息
    1. 站点访问统计
    2. 站点广告统计
    2. 站点采集统计信息
3. 外链模块
    * 外链模块
    1. 外链表
4. 用户系统模块
    * 全局用户系统，隔离不同的系统
    1. 用户表
    2. 用户数据表
5. 域名DNS模块
    * 配置不同的域名，全局域名配置，处理godaddy域名配置，cloudflare或dnspod域名配置下发，cdn域名下发
    1. 域名表
    2. 厂商表
    3. CDN业务表
    4. DNS表
6. 采集模块
    * 全局采集模块，需要处理不同的业务系统的采集，相应的配置都存储在这里
    1. 采集配置表
    2. 采集任务表
7. 前端业务配置
    * 前端API、域名等组合配置，基本系统配置
    1. 前端业务配置表
8. 站点地图
    * 为每个不同的域名前端等提供不同的站点地图
9. 三方API配置
   1. 代理
    * 为采集业务提供全局代理IP，此处需要接入第三方采集系统
   2. 翻译
    * 第三方翻译模块，用于采集海外站点，或者是采集不同的系统
   3. 域名
    * 第三方域名平台
   4. 验证码
    * 第三方验证码识别，提供采集需登录情况处理
10. SEO业务配置
    * 全局站点SEO配置，如同义词，链接方式等
11. 业务分类库
    * 业务分类库
12. 业务库
	1. 资讯库
	    * 资讯类型
	2. 电商库
	    * 电商类型
	3. 体育库
	    * 体育类型
	4. 彩票库
	    * 彩票类型
	5. 游戏库
	    * 游戏类型
	6. 小说库
	    * 小说类型
	7. 视频库
	    * 视频类型
	8. 其它类
	    * 其它类别，如图片、旅游
13. 搜索业务
